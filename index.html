Ideas to get started
  • Safe path: Find the best lit route from point A to point B (use google maps/data on where street lights are). (Here is an example implementation by UVic Web Dev members Christophe Pouliot, Dominique Charlebois and AJ at nwHacks.)
  • Pet a Pooch: Choose a type of dog you would like to pet and it get recommendations for which dog park you are most likely to find it (could use location of user, cross-references with dog licenses by neighbourhood, etc.)
  • Conquest for Liquid Courage: Find all places in walking distance with a liquor license
  • Unified Garbage Collection Scheduler: Put garbage collection schedules for Saanich and Victoria in the same place (exports to google calendar?)

Judging

Technical Complexity /20 (weighted)

How technically difficult was the project to build and execute? How well was it executed? Were there any obvious bugs in the demo? Judges will briefly review the source code and assign a score. This category will be weighted by the experience of the team to make the playing field even for first year students competing against 4th years or professionals.

Presentation /10

At the end of the hackathon each team will give a short presentation and demo of their project. The presentation should include:

  1. A brief description of the motivation to build the project
  2. An overview of technologies used
  3. A demo of how the project works, and the scenario where it would be used
  4. An overview of what the nexts steps for the project should be, are there features that need to be added? Is there a minor bug that needs to be fixed? What would the team need to do after the hackathon in able for the project to be ready to be used by the people of Victoria.
Did You Demo a Live Version? /10
  • 0/10 - Didn't have a working project
  • 5/10 - Had a live demo, but not something that everyone could try
  • 10/10 - Had a live demo, everyone could try your project
Bonus: Benefit to the Community /5

For bonus points, the judges will assign a score based on how beneficial they think the project will be to the broader Victoria community. 1 being the project would have a marginal impact, and 5 being the project has the potential to be used daily by many Victorians.
